# ex46

>
	建立一个项目骨架, 这个骨架具备让项目跑起来的所有内容。新建项目时候，只要把这一节的代码复制过去，修改就行了。
	包括:

		项目文件结构
		自动化测试代码
		模组
		安装脚本


需要安装一些依赖的python包，包的作用下节再说:
	
	pip
	distribute
	nose
	virtualenv
	
如果不用系统依赖的包管理工具，如 brew 和 apt-get。可以用 easy_install 安装 python 包。例：

	`easy_install pip`
	`pip install PIL`

所有python包都在 http://pypi.python.org 网站注册。
